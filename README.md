
# Portfolio Website
Second iteration of yeasirhugais.com. Built with React & hosted on Oracle Cloud.

## Installation
Clone Repo

CD into directory

```
npm install
```

```
npm start
```

or

run in docker

CD into directory

```
docker-compose up -d
```
## Frameworks & Library
* React/JSX
* Javascript
* CSS
* Bootstrap
* Animate.css
* Owl Carousel
